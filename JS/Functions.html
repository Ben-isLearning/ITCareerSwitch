Functions

a Function is the term for a bundle (large or small) of code that is reusable
Functions Execute tasks or perform calculations
Functions are simply blocks of code that designed to perform particular tasks.

Part 1; Functions need to be declared, created and given a purpose
Part 2; Functions need to be 'called' (or Invoked) by something in a different space

Part 1 is the Setup, Part 2 is the Execution.


Anonymous Functions

Are functions that use the function keyword, don't have a name, don't take any values,
They just have keyword and parenthesis
Example: Declare a function, Set its Purpose... Then call it.

Step 1: Create
let show = function () {
console.log("Anonymous Function"); };
Step 2; Call
show();


Named Functions

Are functions that use the function keyword, a meaningful, descriptive name, have paramenters (Thus take arguements) and have parenthesis.
These functions are meant to be reusable snippets of goodness.

Scenario: You're a trying to sell cheap rubbish as real Addidas goods.
Example; (Paste the Bolded text into the Console, Block by block)

(Function Declaration) (Function Name(FunctionParameter)) {
(Parameter).(Attribute) (Set to) "Addidas";

function myAddidasRebrander(object) {
object.brand = "addidas";
}

 Declare the Object you wanna work on ~ Knockoff Chinese goods

let trackSuit = {
brand: "chineseRubbish",
year: 2020
};

 Now... (Call / Invoke) Run 'myAddidasRebrander' function on the trackSuit object...
(Arguement = Different value to what function was declared with)
(Function Name (FunctionArguement))


myAddidasRebrander(trackSuit);

Then lets double check what's happened

console.log(trackSuit);

Hey Presto - Just like that we've got a top notch Addidas branded trackSuit!
