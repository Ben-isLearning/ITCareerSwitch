Anonymous Functions

Are functions that use the function keyword, don't have a name, don't take any values,
They just have keyword and parenthesis
Example: Declare a function, Set its Purpose... Then call it.

Step 1: Create
let show = function () {
console.log("Anonymous Function"); };
Step 2; Call
show();

Example: This is Janky AF

console.log(x);
let z = x(4,3);
console.log(z);

Invoking Functions

Functions can be Invoked (or Called) by an event
User clicking a button
Invoked by other JS Code
Automatically / Self-invoked

The return Keyword defines the value that is "returned"
back to the caller, and causing the function to stop

Example: Here, the function will return the product of a * b
function sumMultiplication (a, b) {
return a * b;
}


Example; Whereas here the same function is called,
but the returned value will be stored as the variable X
let x = sumMultiplication(4, 3);

The code has been Executed once, Now we are able to use the output data (Variable X)
Elsewhere in our code / website

We can take it a step further by including our function in an expression.

let z = sumMultiplication(4,3) * 2;


Best Practice; Declare before Invoking / Calling

Define your Functions in the Js file, then invoke them later.
If you attempt to invoke something before it is declared, It will go boom.

Functions can be declared Globally, or Nested inside other functions.

Example 1: Calculating the Square of something

function square(x) {
return x * x;
}
Invoke it to see it working;
square(6);


Functions with Conditional Logic

Conditional logic inside of functions is a fantastic way to check input
To ensure that the functions aren't fragile.
Example;

function multiply (x,y) {
return x*y;
}

Checking...
multiply (4,3);
multiply (4);
multiply ("alpha", "beta");

This works great if you give it numbers.
But if you give in only one number, or anything but a number - it breaks!
Lets bring in the conditional logic...

This function checks if the type of X or the type of Y is Not equal to number
Before doing any further calculations

function smartMultiply (x,y) {
if (typeof x != 'number' || typeof y != 'number') {
return "Invalid Input";
} else return x * y;
}
Checking...
smartMultiply (4,3);
smartMultiply (4);
smartMultiply ("a",3);
smartMultiply (4,"2");


Functions with Multiple Arguements

Create a function to return the maximum value inside a collection of values.
This function declares max = 0, to be worked with later in the function
This function has a For Loop, for the length of the arguments its given
The conditional logic determines if the index of arguements is > max
Then max should equal the value at that index, otherwise no action
When loop has finished, return max (Which has been set to the highest value)

function findMax() {
let max = 0
for (i = 0; i < arguments.length; i++) {
if (arguments[i] > max) {
max = arguments[i];
}
}
return max;
}

Then Call it with various numbers

x = findMax(1, 2, 30, 5, 700, 101, 22);


Create a function to return the total sum of all the given arguements.
This fuction declares sum = 0 to be worked with later ion the function
This function has a for loop, for the length of the arguements its given
The addition/assignment operator keeps adding the value of the index of arguements
When the loop has finished, Everything has been added, then sum is returned

function sumAll() {
let sum = 0;
for ( i = 0; i < arguments.length; i++) {
sum += arguments[i];
}
return sum;
}

 Invoke / Call the function to see if it works...
sumAll(1, 4, 5, 1, 3, 5, 6, 10, 20);

Declare a variable, so that the total can be worked with later down the line...
let total = sumAll(1, 4, 5, 1, 3, 5, 6, 10, 20);
