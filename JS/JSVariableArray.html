Arrays [  ]

Simply; A variable that contains multiple values + Multiple Types
(There is no theoretical Limit)
A BOX full of items

How to Declare

Declare [our array / variable] = [ value1, value2, ...];
let shoppingList = [ "eggs", "milk", 12, 7, "bread"];

Can Display information from an array by index;
console.log(shoppingList[0]); (Would show 1st item in above Array)
console.log(shoppingList) (To Double check)
console.log(shoppingList[1]); (Would show 2nd item in above Array)
console.log(shoppingList[2]); (Would show 3rd item in above Array)

How to Override / Replace

Override / Replace Item in an Array
shoppingList[0] = "bacon"; (Will override/replace the 1st Item)
console.log(shoppingList) (To Double check)

How to Add

Add item to an Array
shoppingList[5] = "sausage"; (Will add a 6th item @ index 5)
console.log(shoppingList) (To Double check)

Properties of Arrays

A Property is simply an attribute that defines infomration about an array
Such as the length of an array
For Example;
Declare (Our Array) = [ value, value, ...];
let thisArray = [1, 2, 3, 4, 5];
Log to Console((Our array).(Property));
console.log(thisArray.length);

Methods of Arrays

A Method is executable code / functionality that belongs to an array or object
Essentially Functions that are associated with Arrays
For Example;
Declare (Our Array) = [ value, value, ...];
let thisArray = [1, 2, 3, 4, 5];
(our Array).(This Method());
thisArray.reverse();
thisArray.join();
thisArray.sort();
