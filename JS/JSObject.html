Objects / Types

Objects as a whole

Simply put, {Objects} are simply variables that contain more variables,
structured from Name:Value pairs, refered to as Properties.
And Functions (Inside the object), refered to as Methods.
Object in Javascript are everywhere! Understand Objects and you understand JS!
Objects have Properties (Attriubutes) and Methods (Actions)
Consider a real life object, a Pizza

All {Pizza}'s have Properties (Attributes);
Toppings
Crust
Size
Etc

They also have Methods(Actions) associated with them too
Freeze
Cook
Prepare
AddMoreToppings

Everything is an Object;
Booleans, Numbers, Strings (Primitives Can be Objects)
Whereas Dates, Math, Regular Expressions, Arrays, Functions Objects Are ALWAYS objects

Creating an Object;

Object literal ~ {}

let thisObject = {
alphaVariable:101,
betaVariable:404,
firstName:"Dolan"
};

2) Keyword new ~ (Bad Practice - Takes So much longer)

let thisObject = new Object();
thisObject.alphaVariable:101,
thisObject.betaVariable:404,
thisObject.firstName:"Dolan"

3) Object Contructor ~ (Lay out Schematics for an object "type"
Then input data to Create various similar Objects)

function familyMember(first, last, age, relation) {
this.firstName = first;
this.lastName = last;
this.age = age;
this.relation = relation;
}

Once the shematics are laid out; Input values for each object;
let myFather = new familyMember("Rick", "James", 50, "Father");
let myMother = new familyMember("Sally", "Rudolph", 21, "Mother");


Properties

Properties are the most important part of an Object
Properties are the parts that make up the whole
Properties can be referred to, added, changed and deleted.
via Dot Notation ~ Object.property
Ex: person.firstName + "is " + person.age + " years old";

Via Bracket Notation ~ Object["property"]
Ex: person["firstName"] + "is " + person["age"] + " years old";

Looping Through Object Properties (For-In Object)

Declare our text variable (Something to be written to) and our object

let text = "";
let thisObject = {
alphaVariable:101,
betaVariable:404,
firstName:"Dolan"
};

Declare a for loop to look for the properties IN the object;
for (i in thisObject) {
text += thisObject[i];
}

The Loop will return "101404Dolan".
Add property:value; pair to an existing Object;
thisObject.deltaVariable = 505;
Delete property:value; pair of an existing Object;
delete thisObject.deltaVariable;

Number

Number is a universal type;
rather than decimal, integer, large integer floating point etc...
let a = 1;
let b = 12.5;
let c = 20.12341
All of these declarations are variables with Number type data.

let delta = "12.5";
This declaration will produce a variable with String type data.
let d = functionCreateNumber(delta);
This declaration will use the d varible as an arguement to
the functionCreateNumber to create a New variable with Number type data
(functionCreateNumber is uncoded, just an example)

If we are expecting number values to be returned from user input,
we can use conditional logic through if with the isNaN function
to double check the input is correct before continuing.

let delta = "12.5";
let phoneNumber = "Sausage Rolls" This declaration will produce a variable with String type data.
*Assuming this is USER INPUT with incorrect format*

if (isNaN(delta)) {
console.log("It's not a number");
}

if (isNaN(phoneNumber)) {
console.log('It's not a number!');
}

This conditional logic will use the isNaN function to
check whether a value is NaN or not
Where JS is a loosely typed language, "12.5" can be either String or Number.
If (delta = "catfish") it certainly would have come up as NaN.

Math Object (Math.xxx)

The Math Object operates much like a calculator inside JS
The object holds a multitude of properties and functions
Math.function(variable, [variables]);
Ex;
let a = 200.6;
let b = Math.round(a);
let c = Math.floor(a);
let d = Math.ceil(a);
B) This rounds our variable to the nearest whole number (201)
C) This rounds our variable UP, no matter what (201)
D) This rounds our variable Down, no matter what (200)


let x = 100;
let y = 200;
let z = 300;
let biggestValue = Math.max (x, y, z);
let smallestValue = Math.min (x, y, z);
max finds the largest variable, whereas min finds the smallest. (300 / 100)

Math.random();
This returns a random number between 0 - 1;
Really usefull for RNG functions like rolling dice.

Math.PI;
This returns the value of Pi

Math.SQRTx;
This returns the Square Root of x

Floating Point Intiger pitfalls;
let x = 0.2 + 0.1;
Will return x = 0.3000000000000000000000004

Whereas let y = (0.2 * 10 + 0.1 * 10) / 10;
Will return y = 0.3

This is Janky AF, but its a work-around.


Strings

Strings are used for storing and manipulating text data.
can be either start and end with " or ', but not a mixture.
This is due to how "Quotes" are represented within Strings
let xx = "He said "its wednesday" today" would only show "he said"
and the rest would break the code;
let a = "This is Fine!";
let b = 'This is Fine!';
let c = "He said 'this is fine' to me";
let d = 'He said "this is fine" to me';

TL;DR Start on Singles, Include Doubles

OR Escape special characters.
let d = "he said \"this is fine\" as well - But looks Janky";

Strings Are Objects;

As such, Strings have Properties and Methods

let exampleString = " I like mud. I want Mud. Bathe in MUD ";
String Properties;
length; - Returns the length of the string
exampleString.length;
 String Methods;
indexOf() - Returns the first Index of the 1st Entry of the search term
exampleString.indexOf("mud");
lastIndexOf() - Returns the first Index of the Last Entry of the search term
exampleString.lastIndexOf("mud");
charAt() - Returns the character at a given index
exampleString.charAt(10);
match() - Searches for and Returns the match of a given value /value/gi (G for global | I for Case Sensitive)
exampleString.match(/mud/gi);
replace() - Replace first instrance of X within string, with Y Given value.
exampleString.replace("mud", "greek yoghurt");
search() - returns the first index of a particular search term
exampleString.search("mud");
slice() - Slices information out of a string, to create a new one with the results (startIndex, stopIndex)
exampleString.slice(0,11);
toLowerCase() - Transform Data to LowerCase
exampleString.toLowerCase();
 trim() - Trims any whitespace from the begining and ending of a string
exampleString.trim();
 toUpperCase() - Transform Data to UpperCase
exampleString.toUpperCase();

let x = "abc"; (Best Practice!)
let y = new String("abc");
x = String Literal | y = String Object
x == y = True BUT x === y = False!
Both string literal and string objects gain access to string properties + Methods.

Initiating a Date ~
new Date()
new Date(milliseconds)
new Date(dateString)
new Date(year, month, day, hours, minutes, seconds, milliseconds)
new Date () Creates a brand new date @ current time.
Whereas the rest produce a date off of the given data.

Declare a variable as a Date; let x = new Date();
After Initiating a Date object, We are able to invoke various Methods to retreive data through the GET functionality;
getDate(); = Get the date (5th or 25th) as a Number
x.getDate();
getDay(); = Get the weekday (0-7) as a Number
x.getDay();
 getFullYear(); = Get the four digit Year
x.getFullYear();
 getHours(); = Get the Hour (24h Time)
x.getHours();
 getMinutes(); = Get the Minutes
x.getMinutes();
getTime(); = get amount of miliseconds from 01/01/1970
x.getTime();
toString(); = Convert all data to String Value
x.toString();
toUTCString(); = Converts date data to GMT in String Value
x.toUTCString();
toDateString(); = Removes Time from data, Shows String Value
x.toDateString();

We are able to invoke various methods to input data into a Variable through the SET functionality; setDate(); = set the date (5th or 25th) as a Number
x.setDate(13);
setDay(); = set the weekday (0-7) as a Number
x.setDay(1);
 setFullYear(); = set the four digit Year
x.setFullYear(2001);
 setHours(); = set the Hour (24h Time)
x.setHours(19);
 setMinutes(); = set the Minutes
x.setMinutes(30);
setTime(); = set amount of miliseconds from 01/01/1970
x.setTime(123123123);
