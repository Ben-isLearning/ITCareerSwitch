The Keywords break; and continue;
Do just that...

the break; keyword will stop the loop for good under specified conditions

the continue; keyword will skip this round (iteration) and begin the next round (iteration)
For Example; Loop from 0 - 10 > Log 3 > Stop (Break;)

for (i = 0; i <= 10; i++) {
if (i === 3) {
console.log(i)
break;
}
}

For Example; Loop from 0 - 10 > Log Number > If odd go to top (continue;)

for (i = 0; i <=10; i++) {
if ( i % 2 != 0) {
console.log(i);
} else continue;
}
