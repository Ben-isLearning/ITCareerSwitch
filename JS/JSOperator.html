Comparison Operators

Used to compare values.
== | === | != | !== | < | > |
if ( 1 == '1') ~ Loose Equality - Looks at Value (True)
if ( 1 === '1') ~ Strict Equality - Looks at Value + Type (False)
if ( x != 5 ) ~ NOT equal - Determines if 1st Value is NOT equal to 2nd
if ( z !== 5 ) ~ NOT equal - Looks at Value + Type ~Value (True)
if ( x < 6) ~ LESSTHAN - Determines If 1st value is SMALLER than 2nd value
if ( x > 6) ~ GREATERTHAN - Determines if 1st value is BIGGER than 2nd value

With Loose Equality;
let a = 6;
let b = "6";
if (a == b) {
console.log("The values are equal! - I dont care about Types! I am Loooose!")
} else {
console.log("Not Equal!")
};

With Strict Equality;
let a = 6;
let b = "6";
if (a === b) {
console.log("The numbers are equal!")
} else {
console.log("These Not Equal! The value may be the same, but they are different types I AM SO STRICT!!!")
