How to add JavaScript to HTML

1) Within the head, add the script tag and set the type="" to "text/javascript",
Then type the Js / Embedded Script
Con - Takes a while to Parse / Load on the DOM
<script type="text/javascript">
Javascript Goes here
<script>

2) Within the Head, Body, wherever you need it. / Inline Script
each <script> will be loaded and ran from top to bottom;

3) BEST PRACTICE / External Script
Do not embed JS directly into the HTML,
Simply a add script the head, and have the source elsewhere!

<script src="scripts/script01.js" type="text/javascript"> <script>
