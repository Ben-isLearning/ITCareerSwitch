<audio> </audio>

used to embed sound content in HTML documents.
audio attributes:
src="" ~ sets source file
type="" ~ Tells the browser what the filetype is (prevent loading to fail)
(Aux Text) ~ set text inside the audio tag with error mesage, just in case
autoplay ~ Boolean Value - Sets audio to play as soon as it is able.
controls ~ Boolean Value - Enables user controls
loop ~ Boolean Value - Sets audio to loop indefinately
preload ~ (none | metadata | auto) ~ How file is received preemptively
none / only download metadata / download all
title="" ~ Accessability purposes - provides title
