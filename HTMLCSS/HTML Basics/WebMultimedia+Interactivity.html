<audio> </audio>

used to embed sound content in HTML documents.
audio attributes:
src="" ~ sets source file | use <source> element for multiples.
type="" ~ Tells the browser what the filetype is (prevent loading to fail)
(Aux Text) ~ set text inside the audio tag with error mesage, just in case
autoplay ~ Boolean Value - Sets audio to play as soon as it is able.
controls ~ Boolean Value - Enables user controls
loop ~ Boolean Value - Sets audio to loop indefinately
preload ~ (none | metadata | auto) ~ How file is received preemptively
none / only download metadata / download all
title="" ~ Accessability purposes - provides a title

for Single file implementation,

<audio controls src="whatever.mp3"> Your browser does not support the audio element. </audio>


For Multiple file implmentation,

<audio controls>
Your browser does not support the audio element.
<source src="alpha.mp3" type="audio/mp3" />
<source src="alpha.ogg" type="audio/ogg" />
</audio>


<video> </video>

used to embed a media player which supports video playback into the document.
video attributes:
autoplay ~ Boolean Value ~ Sets video element to play as soon as it can.
controls ~ Boolean Value ~ Enables user controls for the video
loop ~ Boolean Value ~ Sets video to loop
muted ~ Boolean Value ~ Sets video to play but silently
poster="" ~ Sets a poster/thumbnail to show while the video loads
src="" Source attribute - sets source of video | <source> if multiple sources.
height="" ~ Sets height of video - must be absolute value in px.
width="" ~ sets width of video - must be absoute value in px.
preload="" ~ none | meta | auto | ~ Preemptively load nothing, some, or, all.
title="" ~ breif discription for accessibility.



<video src="video.mp4" controls height="300px" width="400px;">
Your browser does not support the video element.
</video>


CSS3 and Interactivity

Psuedo Classes
A psuedo class is a keyword added onto a selector that specifies a particular state

Ex ~
.example {background-color: blue;}
In its regular state, .example is blue.
.example:hover {background-color: green;}
When its being hovered over (in its hovered state) .example is green.
