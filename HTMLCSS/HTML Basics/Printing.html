CSS styling for print

uses media="" attritbute, with media="print" + media="screen" values.


Print style best practices

Create second CSS sheet; to use when being printed
<link rel="stylesheet" href="something.css" media="screen">
VS
<link rel="stylesheet" href="somethingprint.css" media="print">

Re-style text to be readable and static.

Hide non-essential content (display:none;)
Use font size in pt, rather than px or em. (printed copies dont need to be responsive)
Code background-color to white
Code font to black
remove background images (background-image:none;)

Page Breaks;
page-break-before: or page-break-after:
Values~ always; | avoid; | auto;

Print URLs;
takes two values; CSS Psuedo-element + CSS Content.
Psuedo-elements: ::after | ::before | ::first-letter | ::first-line
