Link Element

The Link element is used to link HTML to other document types, CSS JS Etc
<Link> Requires 2 attributes; The rel (Relationship) and the href (Hypertext Reference)
To understand what is being linked to and where

<Link rel="stylesheet" href="styles.css">


Img Element

The Image element is used to provide a single graphic image on a webpage
Img tags require two attributes; src (Source) and alt (Alternitve Text)

<img src="hello.jpeg" alt="Picture of a welcoming hand"
Additional Attributes; title="" id="" longdesc=""

Meter element

meter element displays a visual gauge using a given range.
This meter has a label element attatched by syncronising the Label's for with the Meter's id
for = id
The attributes for <meter> are
value="" max="" min="" low="" high="" optimum="" id=""

at 10/100

<label for="patience"> Patience: </label>
<meter id="patience" min="1" max="100" low="33 high=66" optimum="75" value="10">
at 10/100
</meter>


Progress Element

<progress> is used to display the progress of a task
This progress bar has a label element attached much like the meter element above
the attributes for <progres> are
max="" value="" id=""

20%

<label for="progress"> Course Progress</label>
<progress id="progress" max="100" value="20"> 20% </progress>


Map Element

the <map> element is used to create clickable area/areas on an image.
<map> requires a single attribute, name="", as well as an <img> with the attribute usemap="#"to work with.
PS: Name="" must be the same as usemap="#blabla" And atleast one <area> element to create a clickable area

The <area> element requires 5 attributes,
title=""; Which shape=""; Which identifies what type of shape to be used
coords="":Which takes 4 (or more) coordinates, as the start(x,y) and end(x,y)
locations of the clickable shape
alt="": To provide an alternative for screen readers
href="": to create the clickable section;
