Structural / Block Elements

Header - First Structure Element - Generally Contains Heading Elements

Regular / Inline Elements

Link Element

The Link attribute is used to link HTML to other document types, CSS JS Etc
<Link> Requires 2 attributes; The rel (Relationship) and the href (Hypertext Reference)
To understand what is being linked to and where

<Link rel="stylesheet" href="styles.css">


Anchor Elements A tags
