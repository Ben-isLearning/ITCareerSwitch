jQuery uses CSS Selectors to latch onto elements.
$('simple-li')

Whereas standard JavaScript latches onto the actual elements.
document.getElementsByClassName("simple-li");

Universal;
$(document)

jQuery Selectors (CSS)

Selectors are used to identify what particular DOM element is to be targeted
by the following codeblock.
CSS : .className { color:red;}
jQe : $(".classname").css("color", "red");
(This would be poor implementation, but serves as a side-by-side comparison )

CSS : * {xxx} (Universal Selector)
jQe : $("*")

CSS : #idSausage {xxx} (Id Selector)
jQe : $("#idSausage")
