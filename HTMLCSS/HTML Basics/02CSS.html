Lesson 02 - Cascading Style Sheets

Methods of Configuration

1) External Styles (Best Practice)
2) Embedded Styles (In the Head)
3) Inline Styles (Most Specific)
4) Imported Styles (From Google Fonts etc)


Selector { Declaration }

Each CSS Element requires a Selector that selects what is to be modified
That can be a html element / class / id
And a Declaration that identifies exactly what property is being changed
by what specific value
Syntax;

selector {
Property: Value;
}

body {
color: white;
background-color: orange;
}





Cheat Sheet

Color: (Changes to color of Text)
background-color: (Changes the color of the background)
