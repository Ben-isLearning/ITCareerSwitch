CSS Cheat Sheet

Color: (Changes to color of Text)
background-color: (Changes the color of the background)
font-size: Sets Size of font
font-weight: Sets Boldness of font
font-style: Sets font Decoration
line-height: Sets line height
text-align: Sets alignment, R2L or L2R
text-decoration: Sets decoration - underline etc
text-indent: Indents first line of text within element
letter-spacing: places spaces between characters
word-spacing: places spaces between words
text-transform: transforms text into something new (capitalize etc)
white-space: (sets rules for whitespace)
text-shadow: Horizontal Vertical Blur Color; 00px 00px 00px Color; Negative goes left or down
