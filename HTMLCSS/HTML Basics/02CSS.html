Selector { Declaration }

Each CSS Element requires a Selector that selects what is to be modified
That can be a html element / class / id
And a Declaration that identifies exactly what property: is being changed
by what specific value;
Syntax;

selector {
Property: Value;
}

body {
color: white;
background-color: orange;
}
