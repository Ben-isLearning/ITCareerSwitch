Configure a Flexible Container (Flexbox) Best Resource Here

All items within a flexbox are 'flexible' - they all have the same size
They all have allocated space.

a flexbox is setup through the display: property

display: inline-flex; - For flexible inline containers

Direction ~ flex-direction:

The direction of flow within a flexbox is configured using the flex-direcition: property sets the directional flow of the child elements.
flex-direction: row; (By Default Flows Left To Right)
flex-direction: row-reverse; (Flows from Right to left)
flex-direction: column; (Flows from Top to Bottom)
flex-direction: column-reverse: (flows from Bottom to Top)

Single or Multiline? ~ flex-wrap:

Whether or not the flexbox spans several lines is figured using the flex-wrap: property
flex-wrap: nowrap; (By default, a flexbox will maintain a single-line display)
flex-wrap: wrap; (Flexbox can be set to wrap over as many lines as it needs)

Display Style MAIN AXIS- justify-content:

the justify-content: property will define the alignment on the main axis.
(How the items in flexbox are displayed, and how white space is displayed, if at all)

justify-content: flex-start; (Default - Position @ Beginning of main Axis)
justify-content: flex-end; (Start @ End of main axis)
justify-content: center; (Center @ Middle of main axis)
justify-content: space-between; (Whitespace between elements, no padding on edge)
justify-content: space-around; (Whitespace between and around elements)

Display Style CROSS AXIS - align-items:

The align-items: property wil ldefine the alignment on the cross axis.
(How the items in flexbox are displayed)

align-items: flex-start; ( Items start at beginning of Cross axis)
align-items: flex-end; (items start and the end of the Cross Axis)
align-items: center; (items are centered at the center of the cross axis)
align-items: stretch; ( (Default - items take up 100% of the space along the main Axis)
align-items: baseline; (items are centered along the cross axis, but retain their size)

Display Proportion - flex:

the flex: property will define the display proportions of the items within the flexbox
the flex: property is shorthand for flex-grow, flex-shrink and flex-basis, as such
it can take 1, 2 or 3 properties.
flex-grow: 2; (means let grow factor = 2 | by Default its 1)
flex-shrink: 2; (means let shrink factor = 2 | Default its 1)
flex-basis: 100px; (Means size = 100px - let Flexing occur after | Default its 0)

flex: "flex-grow";
flex: "flex-grow" "flex-basis";
flex: "flex-grow" "flex-shrink" "flex-basis";

(Whether item1 is meant to be larger than item2 etc)

Give Them some space - gap:

The gap: property explicitly controls the space between flex items.
gap: 10px; (With one value, it controls entire gap top, and sides)
gap 10px 20px; (With 2 values, it controls row, then column)
row-gap: 10px; (or long-hand with 1 value)
column-gap: 20px; (or longhand with 1 value)
