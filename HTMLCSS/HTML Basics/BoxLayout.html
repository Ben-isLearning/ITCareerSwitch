Flexible Box Layout (Flexbox)

Configure a Flexible Container (Flexbox)

All items within a flexbox are 'flexible' - they all have the same size
They all have allocated space.

a flexbox is setup through the display: property

display: inline-flex; - For flexible inline containers
display:

Direction

The direction of flow within a flexbox is configured using the flex-direcition: property
flex-direction: row;


Single or Multiline?

Whether or not the flexbox spans several lines is figured using the flex-wrap: property
flex-wrap: nowrap; (By default, a flexbox will maintain a single-line display)
flex-wrap: wrap; (Flexbox can be set to wrap over as many lines as it needs)



Display Style - justify-content:

the justify-content: property will define the alignment on the main axis.
(How the items in flexbox are displayed, and how white space is displayed, if at all)

justify-content: flex-start; (Default - Position @ Beginning of main Axis)
justify-content: flex-end; (Start @ End of main axis)
justify-content: center; (Center @ Middle of main axis)
justify-content: space-between; (Whitespace between elements, no padding on edge)
justify-content: space-around; (Whitespace between and around elements)
