HTML Tables

Tables consist of;

<table> - To define the table
<th> - To define a header cell in the table (Table Header)
<tr> - To define a row in a table (Table Row)
<td> - To define a cell in a table (Table Data)
<caption> - To define a table caption
<colgroup> - specify a group of one or more columns
<col> - specify column properties within a colgroup element
<thead> - groups the header content in a table
<tbody> - groups the body content in a table
<tfoot> - groups the footer content in a table
